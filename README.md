# 🚂 Train Track System

A train track management system built in C++ using linked lists to manage train wagons and track connections.

## Features
- Add and remove train wagons
- Linked list-based track and wagon management
- Dynamic memory management

## Technologies Used
- C++
- Linked List (Data Structure)
